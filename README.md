

# 荣景文川官网 · Rongjing Wenchuan Official Site

荣景文川：一家以虚拟人格为核心，融合潮流玩具、音乐、影视与视觉小说的科技文化工作室。

> This is the official website of **Rongjing Wenchuan**, a virtual-persona based culture & tech studio.

---

## 项目简介

本仓库用于存放「荣景文川」官方网站的前端代码，用来对外展示：

- 公司定位：虚拟人格科技公司 + 符号经济实验室  
- 业务板块：潮流玩具、AI 音乐厂牌、影视与视觉小说游戏  
- 品牌世界观与核心理念  

网站为纯前端静态站点，可部署到 GitHub Pages、Cloudflare Pages 或任意静态服务器。

---

## 功能与版块

- **首屏（Hero）**：LOGO + Slogan + 品牌介绍  
- **About 荣景文川**：公司定位 / 故事 / 世界观关键词  
- **业务矩阵**：潮玩 × 音乐 × 影视 × 视觉小说 游戏  
- **IP & 项目展示**：虚拟人物、单曲 / EP、剧集 / 游戏 Demo 等  
- **联系与合作**：邮箱 / 社交媒体 / 合作通道  



---

## 技术栈

- **HTML5**：语义化结构  
- **CSS3**：自定义样式 + 响应式布局  
- **JavaScript**：基础交互逻辑  
- **GSAP + ScrollTrigger（CDN）**：滚动与动效
- **部署**： GitHub Pages
